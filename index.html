Робота з елементами списку

Кожен елемент списку має свій індекс і отримувати доступ до будь-якого елемента списку можна через його індекс.
У Python використовується система індексації, починаючи з нуля.

Метод join

Метод .join() повертає строку, яка отримана після з'єднання елементів.
Між елементами списку вставляється роздільник, який рівний тій строці до якої застосовується метод.

Метод split

Метод .split() розділяє основний рядок по роздільнику та повертає список рядків.

Методи списку:

Метод append.
Ця функція додає елемент в кінець списку

Метод insert.
Ця функція додає елемент у вказаний індекс списку.

Метод pop.
Видаляє останній елемент списку. Також можна видаляти елементи по індексах.

Метод remove.
Видаляє елемент за вказаним значенням.

Метод count.
Виводить кількість повторень вказаного значення.

Метод len.
Виводить довжину списку.
